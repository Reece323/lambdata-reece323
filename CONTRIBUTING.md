
# Contributer's Guide

# Installation 

Git clone and navigate to the repo from your command_line:

```sh
git clone REMOTE_ADDRESS
cd /path/to/repo
```

Create / Activate virtual env:

```sh
pipenv --python 3.7
pipenv install #does all packages in Pipfile
pipenv shel
```

Run example script:

```sh
python -m my_lambdata.my_script
```
